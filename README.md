# CorePlanetMusicPlayer 6.0

### <u>*该版本主要功能仍在开发！！！</u>

CorePlanetMusicPlayer（以下简称CorePMP），是一个UWP平台下的开源音乐播放器。

你可以使用CorePMP的代码创建、打造自己喜欢的音乐播放器。

使用CorePMP代码时，别忘了在关于界面展示CorePMP的仓库地址！
祝您使用愉快！
